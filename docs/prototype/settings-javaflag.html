Java Flag Settings

Java Flags:
 
 
 

Java flags are passed to the VM of most Java language tasks.
You can use these flags to increase the amount of memory alotted to a
Java language module if you are experiencing OutOfMemory errors.

For details of potential java flags run
    java -X
at a command line.
