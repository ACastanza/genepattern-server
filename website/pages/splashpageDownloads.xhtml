Download the GenePattern Desktop Client and programming libraries:
Install Desktop Client The Desktop Client provides an easy-to-use desktop application as an alternative user interface for GenePattern. GenePattern administrators: You can modify GenePattern server settings only from the Web Client.
Programming Libraries Java, MATLAB, and R libraries allow programmers to run GenePattern modules from their applications.
Public Datasets Public datasets of gene expression data including all data used in the GenePattern tutorial.
