Welcome to GenePattern


To run an analysis:
  1. Select a module from the list at the left.
    GenePattern displays the module and its parameters in this center pane.

  2. Enter values for the parameters and click Run.
    GenePattern displays the process status page to show analysis progress.

  3. When the analysis completes, click Return to start to return to this home page.
    The analysis and its result files are displayed in the Recent Jobs pane at the right.
To view analysis results, you generally use a GenePattern visualizer:
  1. Click the arrow icon next to the result file.
    GenePattern lists the modules most often used to view or analyze this result.

  2. Select the module with the text viewer or image in the module name.
    GenePattern displays the module and its parameters.

  3. Enter values for the parameters and click Run.
    GenePattern displays the analysis results in the visualizer.
