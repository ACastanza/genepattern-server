Or Install Python Library From PIP

PIP is the official Python package manager. It comes with many distributions of Python. To learn more about PIP click here.

To install the GenePattern Python library from PIP, simply enter the line shown below into your command line. Installing packages from PIP in this way may require administrator privilages (ex: sudo).

pip install genepattern-python
